Detailed Description

A mirror of the ProfileTrack struct to store statistics matching a profiled track and the blocks it contains (thanks to Profile::ProfileBlockResult).

In this mirror, the blocks are packed at the beginning of the array to avoid having to iterate over all the blocks that might not have been used. The number of blocks used in the track is stored in blockCount.
