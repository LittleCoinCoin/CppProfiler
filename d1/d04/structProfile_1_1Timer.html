Estimates the CPU frequency by waiting for a given amount of time.

Of course, the more time you wait, the more accurate the estimation will be. 100ms is a good value to start with.

◆ s_estimatedCPUFreq

Profile::u64 Profile::Timer::s_estimatedCPUFreq = Profile::Timer::EstimateCPUFreq(1000)
static

The estimated CPU frequency.

This value is pre-computed once in the cpp file (so at compile time). It is used to estimate the CPU frequency at runtime. You can also recompute it at runtime by calling SetEstimatedCPUFreq.
