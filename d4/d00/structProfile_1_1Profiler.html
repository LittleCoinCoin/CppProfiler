void Profile::Profiler::Clear ( )
noexcept

Clears the profiler's values as well as all its initialized tracks.

Remarks
This resets even the name. If you are looking to reset the values without changing the name, use Reset. If you are looking to reset the tracks only, use ClearTracks.
